Modular.nvim

Установка

Установите Neovim

modular.nvim ориентирован только на последние стабильные и ночные версии Neovim. Убедитесь, что у вас установлены последние версии, если вы испытываете проблемы.

Установите внешние зависимости

  • Основные утилиты: git, make, unzip, компилятор C (gcc)
  • ripgrep
  • Nerd Font (желательно)
  • Установите npm для TypeScript, go для Golang и т. д.

Установите Modular

ПРИМЕЧАНИЕ: Создайте резервную копию вашей предыдущей конфигурации (если она существует)

mv ~/.config/nvim ~/.config/nvim.bak
                mv ~/.local/share/nvim ~/.local/share/nvim.bak
                mv ~/.local/state/nvim ~/.local/state/nvim.bak
                mv ~/.cache/nvim ~/.cache/nvim.bak

Конфигурации Neovim находятся по следующим путям в зависимости от вашей ОС:

ОС Путь
Linux, MacOS $XDG_CONFIG_HOME/nvim, ~/.config/nvim
Windows (cmd) %localappdata%\nvim
Windows (powershell) $env:LOCALAPPDATA\nvim

Клонируйте modular.nvim

Linux и Mac 
git clone https://github.com/masajinobe-ef/modular.nvim.git
                    "${XDG_CONFIG_HOME:-$HOME/.config}"/nvim
Windows

Если вы используете cmd.exe:

git clone https://github.com/masajinobe-ef/modular.nvim.git "%localappdata%\nvim"

Если вы используете powershell.exe:

git clone https://github.com/masajinobe-ef/modular.nvim.git "${env:LOCALAPPDATA}\nvim"

Постустановка

Запустите Neovim

nvim

Вот и всё! Lazy установит все плагины, которые у вас есть. Используйте :Lazy для просмотра текущего статуса плагина. Нажмите q, чтобы закрыть окно.

Шаги установки Fedora 
sudo dnf install -y gcc make git ripgrep fd-find unzip neovim
Шаги установки Arch 
sudo pacman -S --noconfirm --needed gcc make git ripgrep fd unzip neovim
